================================================================================
BattleSnake Agent — Run Instructions
================================================================================

INSTALLATION
------------
pip install -r requirements.txt

ENVIRONMENT VARIABLES
---------------------
PORT          — Flask server port (default 8000)
MCTS_BUDGET   — MCTS time budget in seconds (default 0.92 for live play, use 0.10 for tournament)
NUM_SEEDS     — Number of seeds per matchup in tournament (default 30)
MAX_TURNS     — Max turns per game in tournament (default 200)
RESULTS_DIR   — Output directory for tournament results (default "tournament_results")

================================================================================
SINGLE-AGENT SERVERS
================================================================================

All agents are available (port defaults to 8000, override with PORT=XXXX):

  python teamname.py                  # Contest agent: MCTS with ALL improvements
  python mcts_main.py                 # MCTS all improvements (same as teamname.py)
  python mcts_vanilla_main.py         # Vanilla MCTS (random rollouts, no improvements)
  python mcts_rave_main.py            # MCTS + RAVE/AMAF
  python heuristic_main.py            # Pure heuristic agent
  python random_main.py               # Random agent

Additional MCTS variants (run via agent_server directly):

  python -c "from agent_server import run_agent; run_agent('mcts_heuristic_rollout')"
  python -c "from agent_server import run_agent; run_agent('mcts_prior')"
  python -c "from agent_server import run_agent; run_agent('mcts_opponent')"
  python -c "from agent_server import run_agent; run_agent('mcts_ucb1_tuned')"

================================================================================
MCTS MODE STRINGS
================================================================================

Mode string          Config description
-------------------  -----------------------------------------------------------
mcts                 ALL_IMPROVEMENTS_CONFIG (recommended contest agent)
mcts_all             same as "mcts"
mcts_vanilla         Vanilla MCTS: random rollouts, no priors, no opponent model
mcts_heuristic_rollout  Heuristic rollouts only
mcts_prior           Heuristic rollouts + prior-guided expansion + PUCT
mcts_opponent        Heuristic rollouts + priors + opponent-aware root scoring
mcts_rave            Heuristic rollouts + priors + opponent model + RAVE/AMAF
mcts_ucb1_tuned      Heuristic rollouts + priors + opponent model + UCB1-Tuned
heuristic            Pure heuristic (no tree search)
random               Uniform random legal moves

================================================================================
LIVE MATCHUPS (using battlesnake CLI or run_game.py)
================================================================================

MCTS vs Random
--------------
  python mcts_main.py
  PORT=8001 python random_main.py
  MATCHUP=mcts_vs_random python run_game.py

Heuristic vs Random
-------------------
  python heuristic_main.py
  PORT=8001 python random_main.py
  MATCHUP=heuristic_vs_random python run_game.py

MCTS vs Heuristic
-----------------
  python mcts_main.py
  PORT=8001 python heuristic_main.py
  MATCHUP=mcts_vs_heuristic python run_game.py

Available matchups: mcts_vs_random, mcts_vs_heuristic, heuristic_vs_random

================================================================================
HEADLESS SIMULATION
================================================================================

Run a quick headless game between two agents (no CLI needed):

  python -c "
  from simulate_game import run_game
  from MCTS import mcts_move, ALL_IMPROVEMENTS_CONFIG, VANILLA_CONFIG

  def mcts_all(gs): return mcts_move(gs, ALL_IMPROVEMENTS_CONFIG, 0.1)[0]
  def mcts_van(gs): return mcts_move(gs, VANILLA_CONFIG, 0.1)[0]

  result = run_game([('MCTS-All', mcts_all), ('MCTS-Vanilla', mcts_van)], seed=42)
  for r in result['results']:
      print(r)
  "

================================================================================
ABLATION TOURNAMENT
================================================================================

Run full tournament (30 seeds × 9 matchups by default):
  python tournament.py

Quick test (10 seeds only):
  python tournament.py --quick

Custom settings:
  NUM_SEEDS=20 MCTS_BUDGET=0.15 MAX_TURNS=300 RESULTS_DIR=my_results python tournament.py

Output files:
  tournament_results/game_records.csv   — one row per game
  tournament_results/ratings.csv        — ELO + TrueSkill per agent

================================================================================
PLOT RESULTS
================================================================================

Generate all plots (reads from RESULTS_DIR):
  python plot_results.py

Output plots in tournament_results/plots/:
  win_rates.png        — Bar chart of win rates ±1 SE
  elo_ratings.png      — ELO + TrueSkill conservative scores
  turns_survived.png   — Violin plot of survival distributions
  ablation_ladder.png  — Win rate of each improvement over its baseline
  win_matrix.png       — Pairwise win rate heatmap
  elo_progression.png  — ELO over time for 4 key matchups
  survival_boxplot.png — Box plot of turns survived per agent

================================================================================
MCTS IMPROVEMENTS SUMMARY
================================================================================

Improvement 1 (Expansion)  — Prior-guided expansion: moves ordered by heuristic
Improvement 2 (Selection)  — PUCT: UCB1 + decaying heuristic prior bonus
Improvement 3 (Selection)  — Opponent-aware root scoring: minimax blend
Improvement 4 (Simulation) — Heuristic rollouts: guided, not uniform random
Improvement 5 (Backprop)   — RAVE/AMAF: rapid action value estimation
Improvement 6 (Selection)  — UCB1-Tuned: variance-aware exploration

All improvements are controlled by MCTSConfig dataclass in MCTS.py.
